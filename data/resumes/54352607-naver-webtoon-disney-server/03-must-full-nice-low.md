# 이도현 — 서버 개발자

## 자기소개
사내 시스템을 안정적으로 굴리고 꾸준히 개선하는 백엔드 4년차입니다. 반복 작업이 보이면 개선안을 먼저 제안해 도입하는 편이고, 물류센터가 새로 열려 업무 규칙이 바뀔 때마다 도메인을 유연하게 확장하며 대응해왔습니다.

## 경력

### 한빛물류 · 서버 개발자 (2020.04 ~ 재직 중)
국내 물류회사의 운영 시스템(주문·재고·정산) 서버를 담당하고 있습니다.
- Kotlin·Spring Boot로 사내 운영 API를 구축·운영합니다.
- 새 물류센터가 열릴 때마다 업무 규칙이 바뀌어서, 도메인을 유연하게 확장하며 대응해왔습니다.
- 매달 반복되던 정산 배치와 리포트 작업을 자체 AI Harness로 자동화하자고 팀에 먼저 제안해 도입했고, 야근이 눈에 띄게 줄었습니다.
- 인프라는 Nginx·Redis·Batch·Kafka로 구성하고 Docker로 배포합니다.
- 얽혀 있던 정산 로직을 모듈 단위로 쪼개 재사용성을 높이는 리팩토링을 주도했습니다.
- 운영 지표와 오류를 보는 모니터링 대시보드를 붙여 상시 확인합니다.

## 기술
Kotlin · Spring Boot · Kafka · Redis · Nginx · Batch · Docker · MySQL

## 그 외
- 국내 사내 서비스만 다뤄서 글로벌·현지화 경험은 없습니다.
- 트래픽이 큰 서비스보다는 정확도가 중요한 사내 업무 시스템 위주로 일해왔습니다.
- 프론트엔드는 거의 만져본 적이 없습니다.
