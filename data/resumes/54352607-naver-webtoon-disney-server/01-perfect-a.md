# 정민우 — 서버 개발자

## 자기소개
백엔드 4년차입니다. "AI를 옆자리 동료처럼 쓰자"가 지론이라, 반복되는 개발 과정에서 비효율이 보이면 시키기 전에 먼저 자동화하는 편입니다. 다니던 회사가 핀테크로 피벗하던 격변기에도 낯선 결제 도메인을 자처해 맡으며, 급변하는 상황에서 문제를 붙들고 끝까지 파고드는 일을 즐깁니다.

## 경력

### 리버스페이 · 서버 개발자 (2021.03 ~ 재직 중)
원래 O2O 배달 중개로 시작한 회사인데, 재직 중에 간편결제(핀테크)로 피벗했습니다. 그 격변기를 팀과 함께 넘겼습니다.
- Spring Boot로 결제 서버를 처음부터 설계·구축하고, PG사 세 곳 결제 연동을 직접 개발했습니다.
- 피벗으로 트래픽 성격이 완전히 바뀌어서, Kafka로 결제 이벤트를 비동기 처리하고 Redis로 중복 결제를 막았습니다. Nginx·Batch·Object Storage도 이때 붙였습니다.
- 결제가 몰리는 시간대에 같은 주문이 두 번 승인되던 동시성 버그를 분산 락으로 잡고, 이후 대용량 트래픽을 견디도록 아키텍처를 개편했습니다.
- 흩어져 있던 빌드·리뷰 과정을 자체 AI Harness로 묶었습니다. PR을 올리면 AI가 1차 리뷰하고 테스트를 돌리게 만들어, 팀 배포 리드타임을 절반으로 줄였습니다.
- 결제 장애가 몇 번 있었는데, 그때마다 원인을 끝까지 추적해 회고를 문서로 남기고 재발 방지 알림을 붙였습니다.

### 그린카트 · 백엔드 개발자 (2019.06 ~ 2021.02)
이커머스 상품·재고 도메인을 담당하며 서버 개발을 처음 배웠습니다. 글로벌 셀러 입점 기능을 만들며 다국어·다통화 처리를 경험한 게, 지금 글로벌 서비스에 관심을 갖게 된 계기입니다.

## 기술
Kotlin/Java · Spring Boot · Kafka · Redis · Nginx · Docker/Kubernetes · MySQL
프론트가 깊진 않지만, 사내 어드민을 React로 직접 만들어 써서 읽고 고치는 정도는 합니다.
