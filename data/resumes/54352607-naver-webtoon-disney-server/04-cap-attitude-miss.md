# 박준서 — 서버 개발자

## 자기소개
정해진 요구사항을 정확하고 안정적으로 구현하는 서버 개발자입니다. 큰 조직의 검증된 프로세스 안에서 맡은 파트를 흔들림 없이 수행하는 데 강점이 있습니다.

## 경력

### 대성텔레콤 · 서버 개발자 (2017.09 ~ 재직 중)
통신 과금 시스템의 서버 개발을 담당 파트로서 맡고 있습니다.
- Kotlin·Spring Boot 기반 과금 API를 설계 문서에 따라 구현하고 유지보수합니다.
- 대용량 과금 트래픽을 처리하는 시스템이라, 동시성 이슈와 데이터 정합성을 다뤄왔습니다.
- 해외 로밍 과금 연동으로 외국 통신사와 인터페이스를 개발한 경험이 있습니다.
- 결제·정산 연동을 담당하고, 대량 과금 로그는 NoSQL(Cassandra)에 적재합니다.
- 사내 표준 프레임워크와 Docker/Kubernetes 운영 환경 위에서 개발합니다.
- 팀이 도입한 AI Harness 구축 프로젝트에 참여해 리뷰·테스트 자동화를 담당했습니다.
- 운영 모니터링 도구로 지표를 확인하며, 사내 어드민 화면 일부는 React로 만들었습니다.

## 기술
Kotlin · Spring Boot · Kafka · Redis · NoSQL(Cassandra) · Docker/Kubernetes · React(사내 어드민)

## 그 외
- 큰 조직의 정해진 프로세스 안에서 맡은 파트를 안정적으로 수행하는 데 익숙합니다.
- 한 회사를 오래 다닌 만큼 기존 시스템을 깊이 이해하고 있습니다.
