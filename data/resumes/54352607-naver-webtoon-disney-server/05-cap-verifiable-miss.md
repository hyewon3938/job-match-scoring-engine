# 이도경 — 백엔드 개발자

## 자기소개
백엔드 4년차입니다. 새로운 문제를 만나면 스스로 파고들어 팀에 방향을 먼저 제안하는 편이고, 다니던 회사가 커머스에서 헬스케어로 사업을 틀던 시기에도 유연하게 적응하며 핵심 서버를 다시 설계했습니다. AI를 개발 과정에 적극 붙여 반복 작업을 자동화하는 데 재미를 느낍니다.

## 경력

### 케어브릿지 · 백엔드 개발자 (2021.03 ~ 재직 중)
헬스케어 예약·기록 플랫폼의 서버를 담당합니다.
- Node.js·Express·NestJS로 API 서버를 설계·개발하고 운영합니다.
- 예약이 몰리는 시간대의 트래픽을 감당하려고 Nginx·Redis·Kafka로 인프라를 구성하고, 캐시·큐로 부하를 분산했습니다.
- 반복되던 코드리뷰·테스트를 자체 AI Harness로 묶어 자동화하자고 팀에 먼저 제안해 도입했고, 배포 주기를 크게 줄였습니다.
- 회사가 커머스에서 헬스케어로 피벗하던 시기에, 기존 주문 도메인을 걷어내고 예약·의료기록 도메인을 새로 설계하며 급변하는 요구에 유연하게 대응했습니다.
- 지속적인 모니터링으로 병목을 찾아 개선하고, 대량 알림 발송의 동시성 이슈를 큐로 풀었습니다.

### 띵스마켓 · 백엔드 개발자 (2019.02 ~ 2021.02)
중고거래 앱의 상품·거래 서버를 개발했습니다. 첫 직장이었고 서버 개발의 기본기를 익혔습니다.

## 기술
Node.js · Express · NestJS · TypeScript · Nginx · Redis · Kafka · Docker · MongoDB

## 그 외
- 커리어 내내 Node.js 진영에서 일해서, **Kotlin이나 Spring Boot는 실무에서 써본 적이 없습니다.** 다만 JVM 생태계 학습에는 관심이 있습니다.
- 이직 사유는 더 큰 트래픽과 글로벌 서비스를 경험하고 싶어서입니다.
