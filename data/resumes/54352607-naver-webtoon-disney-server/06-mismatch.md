# 강태오 — 백엔드 개발자

## 자기소개
웹 서비스 백엔드를 만들어온 4년차 개발자입니다. 상품·주문·결제까지 커머스 도메인 전반을 다뤄왔고, 최근에는 새로운 서버 기술 스택에도 관심을 넓혀가고 있습니다.

## 경력

### 스타일커머스 · 백엔드 개발자 (2020.05 ~ 재직 중)
패션 커머스 웹 서비스의 서버를 개발·운영합니다.
- PHP(Laravel)로 상품·주문·회원 API를 개발했습니다.
- 데이터는 MySQL 중심이고, 트래픽은 중소 규모라 관리형 클라우드에 대부분 맡겼습니다.
- 결제는 외부 PG의 결제창을 붙이는 수준으로 연동했습니다.
- 코드를 짤 때 ChatGPT를 참고 용도로 씁니다.
- 관리자 페이지는 jQuery와 약간의 React로 만들었습니다.

## 기술
PHP · Laravel · MySQL · JavaScript · jQuery

## 그 외
- 최근 Kotlin과 Spring에 관심이 생겨 토이 프로젝트로 공부를 시작했습니다.
- 서버 개발을 계속 해왔지만, 대용량·분산 환경은 아직 경험이 없습니다.
