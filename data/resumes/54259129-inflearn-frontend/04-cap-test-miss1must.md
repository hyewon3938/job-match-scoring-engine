# 김지원

## 한줄소개
사용자 경험을 최우선으로 생각하며, 다양한 기술 스택을 활용해 웹과 모바일 환경에서의 최적의 솔루션을 제공하는 프론트엔드 개발자입니다.

## 경력 요약
5년 이상의 프론트엔드 개발 경력을 보유하고 있으며, React와 Next.js를 활용한 웹 애플리케이션 개발과 React Native를 통한 모바일 앱 개발에 능숙합니다. 다양한 프로젝트에서 모노레포 환경을 구축하고, Micro Frontend 아키텍처를 운영한 경험이 있습니다. 사용자 행동 데이터를 기반으로 제품 개선에 기여하며, 다국어 서비스 개발에도 참여했습니다.

## 경력 사항

### ABC 테크놀로지 (2018년 6월 - 현재)
**프론트엔드 개발자**

- **웹 애플리케이션 개발**: React와 Next.js를 활용하여 다양한 웹 애플리케이션을 개발하고 배포하였습니다. 특히, 서버에서 미리 렌더링하는 방식으로 사용자 경험을 개선하였습니다.
- **모바일 앱 개발**: React Native를 사용하여 Android 및 iOS 앱을 개발하고, 다양한 디바이스에 최적화된 UI/UX를 구현하였습니다.
- **모노레포 환경 구축**: pnpm과 Turborepo를 활용하여 모노레포 환경을 구축하고, 공용 패키지와 서비스 코드를 효율적으로 관리하였습니다.
- **Micro Frontend 운영**: Module Federation을 활용하여 독립적으로 개발 및 배포 가능한 프론트엔드 아키텍처를 운영하였습니다.
- **다국어 서비스 지원**: i18next와 locize를 활용하여 다국어 서비스를 지원하였으며, 한국어, 영어, 일본어, 베트남어 등 다양한 언어로 서비스를 제공하였습니다.
- **데이터 기반 제품 개선**: Mixpanel과 GTM을 활용하여 사용자 행동을 분석하고, Hackle을 통한 A/B 테스트로 제품 개선에 기여하였습니다.
